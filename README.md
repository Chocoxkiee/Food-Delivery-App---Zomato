# Zomato Website Clone (HTML Project)

This project is a simple front-end clone of the Zomato website developed using HTML.  
It was created as a **group project** to practice basic web development and website structure design.

The project demonstrates a multi-page website with different sections such as menu, login, offers, and payment pages.

## Project Type
Group Project – Front-End Web Development

## Technologies Used
- HTML
- Basic Web Page Layout
- Static Images


## Website Pages
The website contains several pages that simulate features of a food ordering platform.
- **index.html** – Main homepage of the website
- **login.html** – User login page
- **menu.html** – Displays food menu items
- **offers.html** – Shows promotional offers
- **payment.html** – Payment page for orders
- **upi.html** – Payment method interface
- **history.html** – Order history page

## Purpose of the Project
This project was created to:
- Practice building a multi-page website using HTML
- Understand basic website navigation
- Learn how to organize web project folders (images and pages)
- Work collaboratively in a group development environment

## Note
This is a **static front-end project for educational purposes only** and does not include backend functionality or real payment processing.
